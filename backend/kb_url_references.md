## IMPORTANT NOTES FOR BOT / DEVELOPER

1. **URL Freshness:** These URLs were verified as of April 2026. Axis Bank may update URL structures. Re-verify quarterly or when a URL returns a 404.
2. **URL Pattern:** Most Axis Bank card URLs follow: `https://www.axisbank.com/retail/cards/credit-card/[card-slug]/[section]`. Sections are: fees-charges, eligibility-documentation, features-benefits. Do NOT guess slugs for unlisted cards.
3. **No Subdomain URLs:** Avoid URLs from `a2622016.axisbank.com` or other subdomains unless explicitly verified — use main `www.axisbank.com` equivalents.
4. **PDF Links:** PDF URLs (ending `.pdf`) may be moved by Axis Bank without redirects. When citing a PDF, also provide the parent web page URL as a fallback.
5. **RBI CMS Portal:** `https://cms.rbi.org.in` is the ONLY correct URL for filing RBI Ombudsman complaints. Do not cite any other URL for this purpose.
